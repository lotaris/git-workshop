## Aliases

```bash
# Git has no aliases by default.
# If you want to commit with `git ci` instead of `git commit`, for example,
# you can use the alias command.
git alias ci commit

# This will add the alias to your ~/.gitconfig file.
cat ~/.gitconfig
```
